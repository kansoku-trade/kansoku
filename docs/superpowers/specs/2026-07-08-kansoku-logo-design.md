# Kansoku Logo Design

## Decision

The project identity is **Kansoku**, with a restrained Japanese science-fiction observatory tone.

The logo uses a **Minimal Observer** symbol:

- Three curved paths represent possible market scenarios.
- One yellow node marks the observed path.
- Yellow is the identity color because it reads as signal, observation point, and alert without requiring a literal trading motif.

## Palette

| Role | Color |
| --- | --- |
| Observation yellow | `#FACC15` |
| Highlight yellow | `#FEF08A` |
| Observatory black | `#080D14` |
| Muted path | `#E2E8F0` at reduced opacity |

## Usage Rules

- Prefer the standalone mark for compact UI surfaces.
- Prefer the lockup for README, splash, and brand headers.
- Use yellow as an accent, not as a full-page background.
- Keep alternate paths muted so the observed path remains the semantic focus.
