# Kansoku 更新日志

每个版本一节，`## X.Y.Z — YYYY-MM-DD` 格式，新版本插在最上面。
发版 CI（desktop-release.yml）会提取对应版本的段落作为 GitHub Release 的说明，
段落缺失会直接让发版失败——先写说明，再发版。

## 0.22.0 — 2026-07-22

- 首页「市场全景」改成方块面积图：自选/持仓和全市场两个视图都能看，块越大代表体量越大（自选持仓按市值算，全市场按板块涨跌幅算），颜色深浅代表涨跌方向和力度
- 事件日历从一条时间线换成月历 + 未来 7 天列表：财报和宏观数据用不同颜色的点区分，自己持仓的财报会加重点标出，今天会高亮，点某一天还能只看那天的事件
- 顶部状态条整合成一行：大盘指数新增道琼斯（.DJI），和市场温度合并显示；复盘板从侧边栏改成点一下弹窗查看，页面更清爽

## 0.21.0 — 2026-07-21

- 打开个股页默认直接进实时盘面：有最新分析报告时自动叠加显示在图上，不再先落到静态分析页
- 高级 K 线标注（自动形态识别、期权墙位置）纳入 Pro：免费版图表保留基础标注，付费解锁后自动出现
- 修复两个分析相关的小问题：图表实时刷新偶尔建立不起来（要重开页面才恢复），以及代码相近的股票（如 DRAM 与 DRAM 期权）可能查到彼此的分析
- 个股页加载中不再是一行文字，换成与实际布局一致的骨架屏，减少闪动
- 前端按页面拆分加载：首屏只加载当前页面的代码，打开应用和切换页面更快

## 0.20.0 — 2026-07-21

- 修复 0.19.2 打包版本里部分付费页面打不开的问题：安装包内网页资源的路径少算了一层，导致付费界面的部分模块加载失败
- 修复从旧版本升级后打开「设置」页直接崩溃的问题：本地缓存的 AI 模型配置比新版少一项，现在会自动补齐缺失项
- 短线预测图新增「二次突破（SB）」结构标注：第一次冲高/破低失败、第二次再攻的位置会直接画在图上，各周期的技术要点里也会写明确认价和触发时间
- 图层面板改版：原来 13 个勾选框换成「精简 / 标准 / 全部」三档预设，想自己调的话展开自定义即可；标注还能在「只看最近 / 看全部」之间切换，切换时不再重新拉数据，反应更快
- 主图和下方 MACD 图的十字光标联动：鼠标在任一张图上移动，另一张会同步定位到同一时刻，对齐价格和指标不用再靠眼估
- AI 设置新增「记忆整理」这个模型角色，可以单独给它挑模型，不再跟着其他角色走

## 0.19.2 — 2026-07-19

- 安装包大幅瘦身：应用核心从 46MB 压缩到约 10MB，下载和自动更新明显更快，启动加载的文件也更少
- 打包方式重构为单一构建，主程序与付费模块共用同一套运行时，消除了一类「打包后模块找不到、付费功能加载失败」的隐患，并在构建阶段加了防泄漏校验

## 0.19.1 — 2026-07-19

- 精简安装包体积：打包时去掉了运行时用不到的源码映射、类型声明和原生模块编译产物，安装包更小、下载更新更快

## 0.19.0 — 2026-07-19

- 激活体验补完：输入授权码激活成功后，应用会弹窗提示付费功能已解锁并一键重启，不再需要手动退出重开；后台校验拿到解锁凭证时同样会提示重启
- 修复部分已发布版本激活授权时请求失败的问题（授权服务端已修复，旧版本无需升级也已恢复正常）
- AI 输出语言口径统一：各 AI 功能（复评、对话、深度研究等）的回复语言更一致，减少中英混杂

## 0.18.0 — 2026-07-19

- 订阅组件改为加密交付：安装包内不再携带可直接运行的付费代码，激活授权后才会解锁装载，防破解能力显著增强；已订阅用户的功能和数据完全不受影响
- 激活流程更新：输入授权码激活成功后需要重启一次应用，付费功能才会生效——界面会明确提示「重启后生效」，不会再出现激活了却不知道为什么没解锁的情况
- 授权激活与校验改走自建的授权服务（kansoku-portal.innei.dev），不再从应用直连支付平台，激活更稳定、重试更友好
- 付费功能的锁定提示进一步细化：能区分「本构建不含付费组件」和「已含付费组件、激活/重启后可用」两种情况，提示语各自准确
