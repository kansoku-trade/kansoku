# Kansoku 更新日志

每个版本一节，`## X.Y.Z — YYYY-MM-DD` 格式，新版本插在最上面。
发版 CI（desktop-release.yml）会提取对应版本的段落作为 GitHub Release 的说明，
段落缺失会直接让发版失败——先写说明，再发版。

## 0.5.1 — 2026-07-14

- 修复打包版「重新分析」失败：以前装好后读不到短线分析纪律文件（intraday-signal SKILL.md），会反复报错并中止；现在安装包自带这套纪律文件，重估可以正常跑

## 0.5.0 — 2026-07-13

- 新增应用内日志查看：菜单「帮助 → 查看日志…」、设置里的诊断入口，以及命令面板搜「日志」；主进程日志会落盘，方便排查 AI 报错
- 菜单新增「帮助」：查看日志、选择数据目录、从 repo 导入数据都挪到这里；App 菜单更清爽
- 右键菜单升级：桌面端用系统原生菜单（输入框/选中文字有复制粘贴等默认项），业务菜单（如标签页）统一走命令层，Web 端仍用应用内菜单
- 可自定义数据目录：打包版可把数据放到自己指定的文件夹，设置页和菜单都能改
- 复盘页 journal 区增加「跑分析」按钮，方便从历史笔记直接触发分析
- 自动分析会带上事件风险信息；生成分析时会校验预测字段；收盘复盘记分板更完整
- 修复全屏页面高度被桌面标题栏顶掉一部分的问题

## 0.4.0 — 2026-07-13

- 自动 AI 重估全面升级，和手动跑 intraday-signal 分析用同一套逻辑：会先查 X 上的实时情绪和突发消息、看期权关键价位（大量期权持仓聚集的价格，容易吸引股价、也容易扫掉止损）、按账户实际资金给仓位建议，并把完整分析（含消息来源和历史命中率记分板）写进当天的 journal 文件
- 修复 AI 重估用 DeepSeek 模型时一直失败的问题（推理档位「最低」被上游拒收，现在会自动换成有效档位）；分析失败时也会显示真实的出错原因，不再只报「未提交预测」
- 新手引导新增第三步：连接 X/Twitter（opencli），带连接状态自检
- 自动更新检查间隔缩短为 1 小时，新版本能更快推送到你手上

## 0.3.0 — 2026-07-13

- 打开还没有 AI 分析的股票时，不再显示空白提示页，而是直接进入实时 K 线图：图表随行情自动更新，右侧的环境、消息、复盘、AI 点评面板照常可用，「预测」位置放了「AI 生成分析」按钮，分析生成完会自动切换到正式分析页
- 新增 ⌘K 命令面板：随时按 ⌘K 快速搜索并跳转到某只股票或某个页面
- 图表内核升级到新版（lightweight-charts v5），并更新了全部依赖

## 0.2.3 — 2026-07-13

- 标签栏支持横向滚动：标签多到放不下时不再互相挤压，可以用触控板或滚轮左右滚动，两端有渐隐提示还有更多标签
- 新建标签按钮改为和标签一致的圆角

## 0.2.2 — 2026-07-12

- 应用内更新弹窗现在会直接显示版本更新说明，并附完整更新记录的链接
- 首次启动引导：修复背景中间的一道接缝，深色背景更干净
- 首次启动引导：修正 Longbridge CLI 安装脚本地址

## 0.2.1 — 2026-07-12

- 设置页改版：顶部四格摘要收成一行状态；配置问题直接标在出问题的位置，不再单独列一块"需处理"
- 模型分配更清爽：自定义模型配置完成后自动收起为一行摘要，点「修改」再展开；琥珀色现在只表示警告，选中状态改用中性高亮
- Longbridge CLI 和 Provider 凭据合并为一张「连接」卡片，右栏不再零散
- 按钮、输入框、下拉选择器高度统一，界面对齐更整齐
- 移除「本机 API」开关：桌面版不再提供 HTTP 接口；需要用命令行工具访问数据的话，在仓库里单独运行 `cd app && pnpm start` 即可

## 0.2.0 — 2026-07-12

- 应用正式更名为 **Kansoku**：应用名、菜单、窗口标题全部统一
- 首次启动引导换上 Kansoku 品牌视觉与深色界面
- ⚠️ 因应用标识随更名变更，0.1.1 无法通过应用内自动更新升级到本版——请到 [Releases](https://github.com/Innei/kansoku/releases) 手动下载安装一次，之后的版本恢复自动更新

## 0.1.1 — 2026-07-12

Kansoku 桌面版首个公开版本（macOS · Apple Silicon）。

- 本地图表应用：K 线、成交量与常用指标一屏呈现，支持标签页多图切换
- 接入长桥账户：实时行情推送，盘前、盘后、夜盘时段都能看
- AI 分析对话：在图表旁直接向 AI 提问、生成走势推演，支持接入多家模型服务（含 LobeHub 云端）
- 首次启动引导：一步步完成长桥凭证和 AI 模型配置
- 应用内自动更新：新版本发布后会自动提醒并安装（Sparkle）
- 原生 macOS 体验：系统菜单、内嵌标题栏、窗口状态记忆
